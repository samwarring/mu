Detailed Description

Sets format_options for the stream.

Quantities inserted in the stream will be formatted according to the provided format_options.

This is a stream manipulator with one argument. Clients use this method by inserting an instance into a stream. For example:

mu::format_options fmt = { ... };
std::cout << mu::stream::format(fmt) << my_quantity;
