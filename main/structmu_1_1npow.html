Detailed Description

template<std::intmax_t BaseValue, std::intmax_t Exponent>
requires (BaseValue != 0)
struct mu::npow< BaseValue, Exponent >

Represents an integer base raised to an integer power.

This enables represntation of scales that are too large or too small to fit in a std::ratio.

Template Parameters
BaseValueThe base of the exponential expression. This value cannot be 0.
ExponentThe exponent.
See also
mu::pow2 and
mu::pow10 for convenient aliases of this template. If a user wants an exponential value whose base is not 2 or 10, they can use npow directly.
