Detailed Description

Concept is true if a quantity<FromRep, FromUnits> can be stored in a quantity<ToRep, ToUnits> without scaling the contained value, and without loss of precision.

For the operation to not require scaling, the units FromUnits and ToUnits must be equivalent.
