Detailed Description

Concept is true if scaling FromRep by a Scale can be stored in a ToRep without loss of precision.

Such capability is determined by the rep_traits<ToRep> specialization.

Template Parameters
FromRepThe original rep type holding the pre-scaled value.
ScaleThe type of the scale factor (may be integral or floating point). Note the scale type itself may be "relaxed" prior to multiplication.
ToRepThe resulting rep type that holds the post-scaled value.
