Detailed Description

template<units FromUnits, units ToUnits>
struct mu::detail::analysis< FromUnits, ToUnits >

analysis objects perform dimensional analysis between two unit expressions and determine their relationship to each other.

The analysis itself is performed in the constructor, so the results are always available from a constructed analysis object.

This is a constexpr class by design, so the dimensional analysis can be performed at compile-time.
