Detailed Description

template<units FromUnits, units ToUnits>
struct mu::detail::units_conversion< FromUnits, ToUnits >

A template struct that defines the multiplier needed to convert FromUnits to ToUnits.

The value may be an integer, or a floating point value.

If the conversion value is an integer, this struct stores the value using the narrowest possible type. If the conversion value is not an integer, the struct stores the value as a long double.
